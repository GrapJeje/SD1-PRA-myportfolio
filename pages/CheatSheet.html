Micro-Bit

Forever
Het forever-blok wordt gebruikt om code oneindig vaak te herhalen. Zodra de code in dit blok start, blijft het doorgaan zonder te stoppen, totdat het programma wordt beëindigd.
Show Leds
De functie show leds zorgt ervoor dat er een bepaald patroon van leds wordt weergegeven op het apparaat. Dit kan een vaste afbeelding zijn of een bewegend patroon.
On Shake
De functie on shake detecteert wanneer het apparaat geschud wordt en voert vervolgens de code uit die binnen dit blok staat. Dit wordt veel gebruikt in interacties waarbij beweging wordt geregistreerd.
Show Number
De show number-functie laat een nummer op het scherm zien. Dit is handig wanneer je bepaalde numerieke gegevens of berekeningen wilt tonen.
Pick Random
De pick random-functie selecteert willekeurig een getal tussen twee opgegeven grenzen. Dit is nuttig in scenario's waarbij je willekeurige keuzes wilt maken, zoals bij spelletjes.
If Then Else
De if then else-structuur wordt gebruikt om beslissingen te nemen in de code. Als een bepaalde voorwaarde waar is, wordt het then-deel uitgevoerd, anders het else-deel.
Input Button A
De input button A-functie detecteert wanneer de knop A op het apparaat is ingedrukt en voert de code binnen het blok uit.
Input Button B
De input button B-functie werkt op dezelfde manier als knop A, maar reageert wanneer knop B wordt ingedrukt.
Pause
De pause-functie stopt de uitvoering van de code voor een opgegeven aantal milliseconden, wat handig is om tijd te creëren tussen acties.
Music Play Melody
De music play melody-functie speelt een vooraf gedefinieerde melodie af. Deze kan worden aangepast om verschillende tonen of liedjes te spelen.
Set Brightness
De set brightness-functie stelt de helderheid van de leds in. Hoe hoger de waarde, hoe feller de leds.

HTML

HTML
HTML (HyperText Markup Language) is de opmaaktaal die wordt gebruikt voor het maken van webpagina's. Het definieert de structuur van de inhoud door middel van elementen zoals koppen, paragrafen, links, en afbeeldingen.
DOCTYPE
De DOCTYPE-verklaring vertelt de browser welke versie van HTML wordt gebruikt in het document. Bijvoorbeeld: <!DOCTYPE html> specificeert HTML5.
Head
Het head-element bevat metadata over het document, zoals de titel, links naar stylesheets, scripts, en andere informatie die niet direct zichtbaar is op de webpagina.
Body
Het body-element bevat de zichtbare inhoud van de webpagina, zoals tekst, afbeeldingen, video's, en meer. Alles wat je ziet op een webpagina bevindt zich binnen het <body>-element.
Anchor (a)
Het a-element (anchor) wordt gebruikt om hyperlinks te maken. Hiermee kun je naar andere webpagina's of bronnen linken. Bijvoorbeeld: <a href="https://example.com">Klik hier</a>.
Image (img)
Het img-element wordt gebruikt om afbeeldingen weer te geven. Je specificeert de bron van de afbeelding met het src-attribuut. Bijvoorbeeld: <img src="image.jpg" alt="Afbeelding">.
Div
Het div-element is een container die wordt gebruikt om content te groeperen. Het heeft geen specifieke opmaak, maar wordt vaak gebruikt in combinatie met CSS om delen van de pagina te stylen of lay-outs te creëren.
Form
Het form-element wordt gebruikt om een interactief formulier te maken waarin gebruikers gegevens kunnen invoeren en verzenden. Bijvoorbeeld: <form>...</form> bevat velden zoals tekstinvoer, knoppen en selectievakjes.

CSS

CSS
CSS (Cascading Style Sheets) is een stijltaal die wordt gebruikt om de presentatie van HTML-documenten te beschrijven. Hiermee kun je de lay-out, kleuren, lettertypen en andere visuele aspecten van een webpagina aanpassen.
Selector
Een selector is een patroon in CSS dat wordt gebruikt om te bepalen op welke HTML-elementen de stijlen van toepassing zijn. Bijvoorbeeld: p { color: red; } past de stijl alleen toe op paragrafen.
Class Selector
De class selector wordt gebruikt om stijlen toe te passen op een specifiek element of een groep van elementen met dezelfde klasse. Bijvoorbeeld: .classname { ... }.
ID Selector
De ID selector wordt gebruikt om stijlen toe te passen op een enkel element met een uniek ID. Je gebruikt de ID selector met een hashteken (#). Bijvoorbeeld: #idname { ... }.
Box Model
Het box model beschrijft de ruimte die een element inneemt op de pagina. Het bestaat uit vier onderdelen: content, padding, border, en margin.
Flexbox
Flexbox is een lay-outmodel in CSS waarmee je flexibele en responsieve lay-outs kunt maken. Het biedt eenvoudige methoden voor het uitlijnen en distribueren van ruimte tussen items in een container.
Grid
CSS Grid is een tweedimensionaal rasterlay-outsysteem waarmee je complexe lay-outs kunt maken met rijen en kolommen. Het biedt controle over hoe items binnen het raster zich gedragen.
Media Queries
Media queries worden gebruikt om stijlen te definiëren die afhankelijk zijn van de schermgrootte, zodat een website responsief kan zijn. Bijvoorbeeld: @media screen and (max-width: 600px) { ... }.
Position
De position-eigenschap in CSS bepaalt hoe een element wordt gepositioneerd ten opzichte van zijn normale positie of zijn oudercontainer. De waarden kunnen zijn: static, relative, absolute, fixed, en sticky.
